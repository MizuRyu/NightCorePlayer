# NightCore Player プライバシーポリシー
（最終更新：2026年3月23日）

## 1. 基本方針

NightCore Player は、ユーザーのプライバシーを尊重し、本アプリで取り扱う情報の内容と利用目的をできる限り明確にします。本ポリシーは、本アプリにおける情報の取扱いを説明するものです。

## 2. 本アプリが取り扱う情報

本アプリでは、以下の情報を取り扱います。

| 取得する情報 | 用途 | 保管・送信方法 |
| --- | --- | --- |
| 検索履歴 | 過去の検索語句を再利用しやすくするため | 端末内の UserDefaults に保存 |
| 再生履歴、再生キュー、再生速度、シャッフル・リピート状態 | 前回の状態の復元、再生機能の提供のため | 端末内の SwiftData に保存 |
| Apple Music 利用に必要な認証状態や再生関連情報 | 楽曲検索、ライブラリ表示、再生機能の提供のため | Apple の API / フレームワークを通じて処理 |
| クラッシュデータ、パフォーマンスデータ | 安定性向上、障害分析、改善のため | Apple の仕組みを通じて収集される場合あり |

- 本アプリは、ユーザーの氏名、住所、電話番号、位置情報等を開発者が直接取得する設計ではありません。
- 本アプリは広告目的のトラッキングを行いません。

## 3. 第三者サービスについて

本アプリは Apple Music および Apple の提供するプラットフォーム機能に依存しています。そのため、Apple のサービス利用に伴う情報の取扱いについては、Apple の利用規約・プライバシーポリシー等もあわせてご確認ください。

- Apple Music の提供機能を通じた認証、楽曲取得、再生処理は Apple の仕組みを利用します。
- クラッシュデータやパフォーマンスデータは、Apple の診断・分析機能や App Store の仕組みを通じて扱われる場合があります。

## 4. 情報の保存と削除

- 検索履歴はアプリ内操作で削除できます。
- 再生履歴はアプリ内操作で削除できます。
- 端末内に保存されたアプリデータは、アプリを削除することで削除される場合があります。
- Apple 側で処理される情報については、Apple の設定およびポリシーに従います。

## 5. 第三者提供

開発者は、法令に基づく場合を除き、開発者が保有する情報を本人の同意なく第三者に提供しません。ただし、Apple Music 等の第三者サービス連携に伴い、Apple に対して必要な通信が行われます。

## 6. ユーザーの選択肢

- 検索履歴や再生履歴は、アプリ内の操作で削除できます。
- Apple による分析・診断情報の取扱いは、端末設定や Apple アカウント設定により管理される場合があります。
